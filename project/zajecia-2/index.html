Lab

Na zajęciach zaprojektujemy grę karcianą (brydż lub poker).
zasady gry w brydża: http://www.zasadygry.pl/brydz...
Napiszemy pierwszy program w java - Na razie używając konsoli, a następnie skompilujemy go: 
javac <nazwa>.java
i uruchomimy: 
java <nazwa>
Zapoznamy się z metodą startową “main” i przećwiczymy działanie parametrów wejściowych : 
public static void main(String[] args) {
    …
}

Omówimy również pierwszą pracę domową! (dokończenie projektu gry karcianej)

Ćwiczenia

Na ćwiczeniach rozwiążemy problem podziału skarbów przez bilba. (zadanie z kolokwium z 2001 roku).

Pierwsza praca domowa (1pkt)

Zaprojektuj w DIA diagram klas reprezentujący grę w pokera. Uwzględnij klasy abstrakcyjne, publiczność/prywatność metod (+, -, #), generalizację i agregację.
